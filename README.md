# Warner Magic PRO

Versión PRO de la landing de Warner Magic.

## Archivos principales
- `index.html` → página principal premium
- `style.css` → estilos responsive
- `script.js` → links de contacto y carruseles
- `servicios.html`, `servicios.css`, `servicios.js` → subpágina de herramientas
